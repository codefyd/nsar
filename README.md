# نظام إدارة مجمع حلق تحفيظ القرآن الكريم

منصة متكاملة لإدارة مجمع حلق تحفيظ القرآن الكريم باستخدام GitHub Pages و Google Sheets و Google Apps Script.

## 🎯 المميزات الرئيسية

### 1. بوابة التسجيل والقبول
- تسجيل طلاب جدد بسهولة
- جمع بيانات الطالب وولي الأمر
- معالجة الطلبات من قبل الإدارة
- قبول أو رفض الطلبات مع إمكانية تحديد سبب الرفض

### 2. بوابة بيانات الطالب
- البحث عن بيانات الطالب برقم الهوية أو رقم جوال ولي الأمر
- عرض بيانات الطالب والحلقة والحفظ
- تحديث المعلومات المسموح بتعديلها

### 3. بوابة العاملين
- دخول آمن للمعلمين والمشرفين والمديرين
- إدارة الطلاب والحلق
- إصدار الإنذارات التعليمية والإدارية
- معالجة الطلبات الواردة

### 4. نظام الإنذارات
- إنذارات تعليمية (مرتبطة بالأداء التعليمي)
- إنذارات إدارية (تأخر، غياب، غياب بعذر)
- عتبات قابلة للتخصيص
- إجراءات متدرجة

### 5. التواصل عبر واتساب
- إرسال الإنذارات عبر واتساب
- قوالب رسائل قابلة للتخصيص
- متغيرات ديناميكية في الرسائل

### 6. الإعدادات المركزية
- إدارة قوالب الرسائل
- تخصيص عتبات الإنذارات
- إدارة المستخدمين والصلاحيات

## 🏗️ الهيكلية التقنية

- **React 19** + **TypeScript**

### الخلفية البرمجية (Backend)
- **Google Apps Script** لمعالجة الطلبات

### الاستضافة
- **GitHub Pages** للواجهة الأمامية
- **Google Apps Script** للخلفية البرمجية

## 📋 متطلبات النظام

### قبل البدء
1. حساب Google
2. جدول بيانات Google جديد
3. حساب GitHub (لاستضافة الواجهة الأمامية)

## 🚀 خطوات الإعداد

### 1. إعداد قاعدة البيانات (Google Sheets)

1. افتح [Google Sheets](https://sheets.google.com) وأنشئ جدول بيانات جديد
2. اذهب إلى `Extensions` > `Apps Script`
3. انسخ محتوى ملف `Code_Updated.gs` إلى محرر Apps Script
4. احفظ المشروع واضغط على زر التشغيل لتشغيل دالة `setupSpreadsheet`
5. وافق على الأذونات المطلوبة
6. انشر السكربت كـ Web App:
   - اذهب إلى `Deploy` > `New deployment`
   - اختر `Web app`
   - اختر `Execute as: Me` و `Who has access: Anyone`
   - انسخ الرابط الذي سيظهر لك

### 2. إعداد الواجهة الأمامية

1. استنسخ هذا المستودع أو انسخ الملفات
2. افتح ملف `client/src/lib/config.ts`
3. أضف رابط Web App URL الذي حصلت عليه في الخطوة السابقة
4. قم بتثبيت المتطلبات:
   ```bash
   pnpm install
   ```
5. شغّل خادم التطوير:
   ```bash
   pnpm dev
   ```

### 3. نشر على GitHub Pages

1. أنشئ مستودع جديد على GitHub
2. ادفع الملفات إلى المستودع:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/your-username/repo-name.git
   git push -u origin main
   ```
3. اذهب إلى إعدادات المستودع > Pages
4. اختر `Deploy from a branch` واختر `main` و `/root`
5. انتظر دقيقة أو دقيقتين لنشر الموقع
